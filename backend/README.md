A smart backend system for task management built with **FastAPI**.  
This API goes beyond basic CRUD by introducing **intelligent task prioritization and organization logic**.

---

## Features

- Create, read, update and delete tasks (CRUD)
- Task status management (todo, doing, done)
- Automatic priority scoring system (in progress)
- Clean architecture (production-style backend structure)
- Environment-based configuration
- Database integration (SQLAlchemy)

---

## Tech Stack

- Python 3.11+
- FastAPI
- Uvicorn
- SQLAlchemy
- Pydantic
- SQLite (default)
